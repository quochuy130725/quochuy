Input và Output trong Java

Cách có thể output dữ liệu ra màng hình

cách để có thể in (output) dữ liệu ra màn hình chúng ta sẽ sử dụng System.out.println()

Code 

            System.out.println("Hello World");

Cách input data vào

cách để có thể nhập dữ liệu vào đó chính là dùng 1 class phụ đó là Scanner. bằng cách import thư viện Scanner.java.ultil

Code 

            Scanner sc = new Scanner(System.in);
            int a = sc.nextInt();
