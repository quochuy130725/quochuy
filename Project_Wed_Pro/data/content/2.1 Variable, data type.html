Variable,data types

1. Từ khóa và Định danh (Keywords and Identifiers):

Từ khóa (Keywords):

Java sử dụng các từ khóa tương tự như ngôn ngữ C, ví dụ: int, float, if, else, class, v.v.
Không thể dùng các từ khóa này làm tên biến hoặc hàm.

Định danh (Identifiers):

Tên định danh có thể bao gồm các ký tự: chữ cái (a-z, A-Z), chữ số (0-9), dấu gạch dưới _, và ký hiệu $.
Không thể bắt đầu bằng chữ số.
Java phân biệt chữ hoa và chữ thường, ví dụ: MyVariable và myvariable là hai định danh khác nhau.

2. Kiểu dữ liệu nguyên thủy (Primitive Data Types):

Java cung cấp các kiểu dữ liệu nguyên thủy, gồm:

  • Số nguyên: byte, short, int, long
  • Số thực: float, double
  • Ký tự: char (dùng để lưu trữ một ký tự Unicode)
  • Boolean: boolean (chỉ có giá trị true hoặc false)

cú pháp khai báo :

Code 

              int myNumber = 10;
              float myFloat = 3.14f;
              char myChar = 'A';
              boolean isJavaFun = true;

3. Toán tử và Biểu thức (Operators and Expressions):

Các toán tử và biểu thức:

  • Số nguyên: +, -, *, /, %
  • Số thực: ==, !=, <, >, <=, >=
  • Ký tự: &&, ||, !
  • Boolean: boolean =, +=, -=, v.v.
Ví dụ:
Code 

            int x = 10;
            int y = 20;
            int sum = x + y; // Kết quả: 30

Thứ tự ưu tiên toán tử:

Toán tử trong ngoặc () được ưu tiên trước.
Tiếp theo là toán tử số học, so sánh, logic, và cuối cùng là gán.

4. Mảng (Arrays):

Mảng là một đối tượng chứa nhiều giá trị cùng kiểu.

Khai báo và khởi tạo:

Code 

              int[] numbers = new int[5]; // Mảng chứa 5 số nguyên
              numbers[0] = 10; // Gán giá trị cho phần tử đầu tiên
Mảng nhiều chiều:
Code 

              int[][] matrix = { 
                  {1, 2}, 
                  {3, 4}, 
                  {5, 6} 
              };
              System.out.println(matrix[1][1]); // Kết quả: 4

5.Nhập/Xuất dữ liệu (Input/Output):

Sử dụng lớp Scanner để nhập dữ liệu:

Code 

              import java.util.Scanner;
      
              public class Main {
                  public static void main(String[] args) {
                      Scanner sc = new Scanner(System.in);
                      System.out.print("Nhập số: ");
                      int num = sc.nextInt();
                      System.out.println("Số bạn nhập là: " + num);
                  }
              }

Sử dụng System.out để xuất dữ liệu:
Code 

              System.out.println("Chào mừng đến với Java!"); // Xuất có xuống dòng
              System.out.print("Hello "); // Xuất không xuống dòng
              System.out.printf("Số %.2f", 3.14159); // Định dạng như C

6. Phong cách lập trình Java (Java Style):

Đặt tên biến, lớp, và hằng số:

  • Biến: Sử dụng lowerCamelCase, ví dụ: studentName
  • Lớp: Sử dụng UpperCamelCase, ví dụ:MyClass
  • Hằng số: Viết hoa và cách nhau bởi dấu gạch dưới _, ví dụ: PI, MAX_VALUE.

Quy tắc thụt lề và ngoặc:

Các dòng mã trong một khối phải thụt vào so với dòng chứa ngoặc { hoặc lệnh mở đầu.

7.Gói (Package):

Khái niệm:

Gói là một namespace để nhóm các lớp và interface liên quan.
Ví dụ: java.util là một gói chứa các lớp tiện ích như: ArrayList , HashMap.

Tạo gói:

Code 

            package myPackage;
    
            public class MyClass {
                public static void main(String[] args) {
                    System.out.println("Đây là lớp trong gói myPackage");
                }
            }

Tham số có thể truyền vào phương thức main thông qua mảng String args[] . Ví dụ:

Chạy chương trình với lệnh: java CommandLineExample Hello World

Output
