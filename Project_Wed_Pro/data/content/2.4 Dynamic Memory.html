Dynamic Memory trong java

1. Khái niệm về Dynamic Memory

Dynamic Memory trong Java liên quan đến việc cấp phát bộ nhớ trong runtime, chủ yếu trên Heap để lưu trữ các đối tượng được tạo bằng từ khóa new.

  • Stack Lưu trữ biến cục bộ và tham chiếu đối tượng.
  • Heap Lưu trữ các đối tượng, quản lý bởi Garbage Collector (GC).

2. Ví dụ

Code 

            Animal cat = new Animal(); // Cấp phát bộ nhớ trên heap
            cat = null; // GC sẽ thu hồi khi cần
