Các lớp Stream cơ bản:

Byte Stream:

  1. Input Stream:
    • FileInputStream: Đọc dữ liệu từ file.
    • BufferedInputStream: Đọc dữ liệu hiệu quả hơn với bộ đệm.
    • DataInputStream: Đọc dữ liệu nguyên thủy (primitive types).
  2. Output Stream:
    • FileInputStream: Ghi dữ liệu vào file.
    • BufferedInputStream: Ghi dữ liệu hiệu quả hơn với bộ đệm.
    • DataInputStream: Ghi dữ liệu nguyên thủy (primitive types).

Character Stream:

  1. Reader:
    • FileReader: Đọc dữ liệu từ file.
    • BufferedReader: Đọc dữ liệu hiệu quả hơn với bộ đệm.
    • InputStreamReader: Chuyển byte thành ký tự (thường dùng với mã hóa).
  2. Writer:
    • FileWriter: Ghi dữ liệu vào file.
    • BufferedWriter: Ghi dữ liệu hiệu quả hơn với bộ đệm.
    • OutputStreamWriter: Chuyển ký tự thành byte.

FileInputStream (Đọc dữ liệu từ file):

Code 

            import java.io.FileInputStream;
            import java.io.IOException;
            
            public class FileInputStreamExample {
                public static void main(String[] args) {
                    try (FileInputStream fis = new FileInputStream("input.txt")) {
                        int data;
                        while ((data = fis.read()) != -1) {
                            System.out.print((char) data);
                        }
                    } catch (IOException e) {
                        e.printStackTrace();
                    }
                }
            }

BufferedInputStream (Đọc dữ liệu hiệu quả hơn với bộ đệm):

Code 

            import java.io.BufferedInputStream;
            import java.io.FileInputStream;
            import java.io.IOException;

            public class BufferedInputStreamExample {
                public static void main(String[] args) {
                    try (BufferedInputStream bis = new BufferedInputStream(new FileInputStream("input.txt"))) {
                        int data;
                        while ((data = bis.read()) != -1) {
                            System.out.print((char) data);
                        }
                    } catch (IOException e) {
                        e.printStackTrace();
                    }
                }
            }

DataInputStream (Đọc dữ liệu nguyên thủy):

Code 

            import java.io.DataInputStream;
            import java.io.FileInputStream;
            import java.io.IOException;

            public class DataInputStreamExample {
                public static void main(String[] args) {
                    try (DataInputStream dis = new DataInputStream(new FileInputStream("datafile.dat"))) {
                        int i = dis.readInt();
                        double d = dis.readDouble();
                        System.out.println("Int: " + i);
                        System.out.println("Double: " + d);
                    } catch (IOException e) {
                        e.printStackTrace();
                    }
                }
            }
